## Get started

1. Install dependencies

   ```bash
   npm install
   ```


2. Create an api key at `https://thecatapi.com/signup`

3. Create a `.env` file in the root of the project and add the following:

   ```
    EXPO_PUBLIC_CAT_API_KEY=your_api_key
    ```

4. Start the app

   ```bash
    npx expo start
   ```

5. Open iOS simulator

   ```bash
   i
   ```
